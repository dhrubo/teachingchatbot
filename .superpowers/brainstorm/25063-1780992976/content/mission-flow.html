Key Design Rules

Rule Detail
3+ concept cards minimum LLM may add more, but never fewer. Pre-authored cards are always shown first.
No LLM for cards Zero LLM calls during concept card phase. All pre-authored.
Challenge mode = overlay Full viewport takeover. Chat input hidden. Exit button returns to mission map.
Help chat always available "Ask Tutor" button during challenge mode opens a chat panel overlay. Student returns to challenge after.
Scoring is local No LLM call for grading, scoring, or results. Pure client-side.
Results show what matters Score, accuracy %, mistakes. NOT XP/streak (those come after, on the mission map).
