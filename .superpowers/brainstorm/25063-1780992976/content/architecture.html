Mission State Machine

HOME → SELECT MISSION → INTRO → LESSON → CARDS → READY_GATE → CHALLENGE → RESULTS → COMPLETE
                                    ↕                    ↑
                           HELP_CHAT  ← tap "Ask Tutor"

Flow Detail

1. HOME — Hero + mission map. Guest: random "Today's Mission". Logged-in: "Continue Mission" with progress.

2. SELECT MISSION — Tap a mission node on the map. Mission loads its pre-authored concept cards and intro.

3. INTRO — Short mission intro screen (title, goals, estimated time). "Start →" button.

4. LESSON — LLM teaches the first concept. Uses existing useChat stream. Chat messages visible.

5. CARDS — 3+ pre-authored concept cards shown as a slide deck. "Next →" to advance. No LLM call.

6. GATE — "Ready for Challenge Mode?" prompt (same as current readiness gate UI).

7. CHALLENGE — Full-screen overlay. Questions 1/5, 2/5, etc. LLM generates via emitChallengeBundle.

8. RESULTS — Score screen: "Mission Complete! 4/5 ⭐⭐⭐⭐☆". Review / Continue / Next mission buttons.

9. COMPLETE — Mission marked done in DB. Student returns to mission map.

HELP_CHAT — At any CARDS/GATE/CHALLENGE step, "Ask Tutor" opens a chat overlay. Student types questions, gets LLM response, then returns to the mission flow.
