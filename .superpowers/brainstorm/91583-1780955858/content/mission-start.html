🎯 Starting a mission — what happens next?

When the student clicks "Start today's mission" or taps a mission card, the SARA dashboard should transition to the chat. How should it work?

A

Immediate teaching

Dashboard fades out, new chat opens with the topic pre-selected. The AI immediately starts teaching the first micro-lesson. Smooth, frictionless.

Example: Clicks "Percentages" → chat opens → AI says "Let's start with finding 10% of a number. What's 10% of 80?"

B

Topic entry gate

Dashboard opens the chat, but the existing TopicEntryOverlay shows first (the full-screen "Start learning →" gate with topic intro). Student sees the topic name and gets a moment to prepare.

Example: Clicks "Percentages" → gate overlay shows "Let's learn Percentages!" with [Start learning] → chat begins

I'd recommend A — the dashboard itself is already a transition moment. Adding another gate would be redundant.
